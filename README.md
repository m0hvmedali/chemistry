# 🧪 Chemistry Lab - نظام تعلم الكيمياء الشامل

## 📱 رابط المشروع المباشر
[https://chemistry-eddff948h-m0hvmedalis-projects.vercel.app/](https://chemistry-eddff948h-m0hvmedalis-projects.vercel.app/)

## 🎯 وصف المشروع

تطبيق تعليمي شامل ومتطور للكيمياء مع مساعد ذكي، مصمم خصيصاً للطلاب العرب. يجمع التطبيق بين التعلم التفاعلي والتنظيم الدراسي المتقدم مع إمكانيات الذكاء الاصطناعي.

## ✨ المميزات الشاملة

### 🌓 نظام الوضع الداكن والفاتح
- **تبديل سلس** بين الوضع الداكن والفاتح
- **حفظ التفضيلات** تلقائياً في المتصفح
- **كشف تلقائي** لتفضيلات النظام
- **دعم كامل** لجميع المكونات والألوان

### 📚 نظام إدارة المواد الدراسية المتقدم
- **📁 مجلدات منظمة** لكل مادة دراسية مع ألوان مميزة
- **📋 جداول دروس شاملة** تتضمن:
  - اسم الدرس (Task name)
  - رابط الدرس (Custom field – URL)
  - ملخص الدرس (Custom field – Text)
  - صور أو مرفقات (Attachments)
  - الحالة (Status: جاري / مكتمل / لم يبدأ)
  - تاريخ الدراسة (Due date)
- **🎯 تتبع التقدم** لكل مادة ودرس
- **📊 إحصائيات مفصلة** للأداء

### ⏱️ مؤقت المذاكرة (Pomodoro Timer)
- **⏰ نظام Pomodoro متقدم**: 45 دقيقة دراسة + 20 دقيقة راحة
- **🔔 إشعارات تلقائية** لتنبيه المستخدم
- **⏱️ تتبع الوقت المستغرق** في كل مادة
- **📈 إحصائيات يومية** شاملة
- **🎯 أهداف وقت الدراسة** قابلة للتخصيص

### 📊 Dashboard رئيسي متطور
- **📈 Widget شامل** يعرض:
  - قائمة المواد (Folders)
  - عدد الساعات المنجزة لكل مادة
  - شريط تقدم نسبة الدروس المكتملة
  - Timeline عام لكل المواد
  - عداد يومي لإجمالي وقت المذاكرة
- **📅 Timeline تفاعلي** لترتيب وتواريخ الدروس
- **🎯 مؤشرات أداء** مفصلة

### 📁 نظام رفع الملفات وتلخيص الدروس
- **📤 رفع ملفات متعددة الأنواع**: PDF, DOC, DOCX, TXT
- **🤖 إنشاء ملخصات تلقائية** للمحتوى
- **✍️ كتابة دروس مباشرة** في التطبيق
- **📂 إدارة وتنظيم** الملفات المرفوعة
- **🔍 بحث في المحتوى** المرفوع

### 🤖 شات بوت معلم متطور ومجاني
- **🎓 تدريب على أي ملف** يوضع في مجلد التدريب
- **📖 قراءة وفهم** محتوى الملفات المرفوعة
- **💡 إجابات دقيقة** بناءً على المحتوى المدرب عليه
- **💰 مجاني 100%** بدون خدمات خارجية مدفوعة
- **🧠 ذاكرة محادثة** متقدمة
- **⚡ استجابة سريعة** ودقيقة

### 📖 مصادر التعلم الخارجية
- **📚 مكتبة شاملة** من المراجع والكتب الإلكترونية
- **🌐 روابط مواقع تعليمية** موثوقة ومتخصصة
- **🎥 فيديوهات تعليمية** مختارة بعناية
- **🔬 أدوات تفاعلية** ومحاكيات كيميائية
- **💡 نصائح للدراسة** الفعالة والتنظيم

### 📱 التوافق الكامل مع الأجهزة المحمولة
- **📱 تصميم متجاوب** يعمل على جميع الأحجام
- **👆 تحسين أحجام النقر** المناسبة للأصابع
- **🔤 خطوط واضحة** ومقروءة على الشاشات الصغيرة
- **🧭 تنقل مبسط** وسهل للهواتف
- **⚡ أداء محسّن** وسرعة عالية

### 🧪 المميزات الكيميائية الأساسية (محفوظة)
- **🔬 جدول دوري تفاعلي** كامل (118 عنصر)
- **⚛️ معلومات مفصلة** لكل عنصر كيميائي
- **🧬 نظام المركبات** الشامل (أحماض، قواعد، أملاح)
- **🎮 اختبارات تفاعلية** ممتعة وتعليمية

## 📦 التثبيت والتشغيل

### المتطلبات
- Node.js 18+ أو Bun
- متصفح حديث يدعم ES6+

### خطوات التثبيت

```bash
# استنساخ المشروع
git clone https://github.com/m0hvmedali/chemistry.git

# الانتقال للمجلد
cd chemistry/chemistry-learning

# تثبيت التبعيات (باستخدام Bun - الأسرع)
bun install

# أو باستخدام npm
npm install

# تشغيل المشروع في وضع التطوير
bun run dev

# أو
npm run dev

# بناء المشروع للإنتاج
bun run build

# أو
npm run build
```
